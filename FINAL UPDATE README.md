# Nusantara-NFT-Engine-V2

**Nusantara-NFT-Engine-V2** adalah dashboard visual berbasis Web3 untuk manajemen aset digital yang menggabungkan estetika **Cyber-Batik** dengan ekosistem **Pi Network**.

## 🚀 Fitur Utama (High Utility)
* **Pi Box Seamless Integration:** Sinkronisasi langsung untuk proses minting dan manajemen aset di jaringan Pi.
* **Multi-Format Support:** Optimal untuk Gambar (High-Res), Musik/Audio, Video, dan Model 3D (Metaverse-Ready).
* **Cyber-Batik Interface:** Antarmuka panel kontrol yang fungsional dengan identitas visual batik Nusantara.
* **Minting Validator:** Pengecekan standar metadata (termasuk batas 140 karakter deskripsi) sebelum proses minting dilakukan.

## 🛠️ Stack Teknologi
* **Network:** Pi Network (PiBox)
* **Storage/DB:** PostgreSQL (Neon.tech)
* **Frontend:** Mobile-optimized Dashboard

## 📂 Struktur Folder
* `/src` : Logika utama engine.
* `/assets` : Koleksi file digital (NFT).
* `/database` : Skrip database (Neon.tech).
